 ## 计算笛卡尔积

 ```js
 function cartesianProductOf() {
  return Array.prototype.reduce.call(arguments, function(a, b) {
    var ret = [];
    a.forEach(function(a) {
      b.forEach(function(b) {
        ret.push(a.concat([b]));
      });
    });
    return ret;
  }, [[]]);
}
 ```

 ### 使用方法
 ```js
 cartesianProductOf(['1','3'],['a','b'])
 ```
